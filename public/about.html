About SkyRC

IRC-like chatrooms powered by BlueSky OAuth

What is SkyRC?

SkyRC is a modern IRC-like chat application that uses BlueSky OAuth for authentication. It provides real-time chatrooms where users can join different channels and communicate with others using their BlueSky identity.

Features

  • BlueSky OAuth Integration - Login with your BlueSky account
  • Real-time Chat - Instant messaging powered by Socket.IO
  • Multiple Chatrooms - Join different channels based on topics
  • Custom PDS Support - Works with custom BlueSky domains
  • Mobile Friendly - Responsive design for all devices
  • Session Management - Secure session handling with timeout

Getting Started

Local Development

  1. Clone the repository
  2. Install dependencies: npm install
  3. Generate OAuth keys: npm run build
  4. Start development server: npm run dev
  5. Visit http://localhost:3001

Deployment

SkyRC can be deployed to various platforms:

  • Railway - Easy deployment with automatic HTTPS
  • Render - Free tier available with custom domains
  • DigitalOcean - Full control with App Platform
  • Vercel - Great for static + serverless functions

Security Features

  • ECDSA key pairs for OAuth authentication
  • Session timeout based on inactivity
  • Rate limiting to prevent abuse
  • Input validation for chatroom names
  • Secure cookie handling

Technology Stack

  • Backend: Node.js, Express, TypeScript
  • Real-time: Socket.IO
  • Authentication: BlueSky OAuth 2.0
  • Frontend: Vanilla HTML, CSS, JavaScript
  • Protocol: AT Protocol (BlueSky)
